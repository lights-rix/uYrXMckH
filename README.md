## 内容介绍

本项目旨在设计和实现一款小学家校一体“作业帮”应用，通过该应用，学生、老师和家长可以方便地进行作业的发布、提交和批改。此外，该应用还提供了丰富的互动功能，如即时消息、通知提醒等，以促进家校之间的良好沟通。通过这个项目，我们希望为小学教育领域带来便捷、高效的解决方案。

## 技术介绍

本项目采用以下技术栈进行开发：

- **语言：** Java
- **使用框架：** Spring Boot
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
